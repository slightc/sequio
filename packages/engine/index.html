@sequio/engine — demos & verify pages

The SDK runtime. Run pnpm dev:engine (from the workspace root) or pnpm dev (from packages/engine), then open a page below — the dev server runs on http://localhost:6173. Video decode/export needs a WebCodecs-capable browser (a current Chrome/Edge). The *-test pages are the browser e2e checks driven by pnpm verify:*.
